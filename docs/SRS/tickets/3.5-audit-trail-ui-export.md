# 3.5 Audit Trail UI and Export

**Status:** Open
**Roadmap:** [SRS_IMPLEMENTATION_ROADMAP.md](../SRS_IMPLEMENTATION_ROADMAP.md) §3.5
**Effort:** 3–5 days
**Depends on:** [1.5 Audit Trail Infrastructure](1.5-audit-trail-infrastructure.md)

## Summary

Implement `/admin/audit` with finalized table schema, filters, detail view behavior, pagination strategy, and export semantics.

## Acceptance Criteria

- [ ] Add `/admin/audit` page and nav entry.
- [ ] Finalize and implement table columns (timestamp/user/action/entity/summary baseline).
- [ ] Finalize and implement filter bar (action, entity, user, date range).
- [ ] Finalize and implement detail view pattern (modal, drawer, or inline).
- [ ] Define and implement export behavior:
  - export respects current filters
  - CSV/XLSX supported
- [ ] Pagination/performance behavior is documented and implemented for large datasets.
